# four-handwritings-of-fe
前端四大手写

## 介绍
前端领域，以手写bind、手写深拷贝、手写 EventHub（发布-订阅）、手写 Promise最为常见，本项目即是对四大手写的一一实现。

## 运行

```shell
yarn        # 安装依赖
yarn start  # 运行browser-sync启动http server
yarn test  # 通过mocha运行测试
```
